# ft_transcendence - READ ME


### 1. **Set up the .env file**
prepare and complete the .env file in the folder srcs/secrets :

 ```bash
VAULT_USER_DB=
VAULT_PSWD_DB=

VAULT_URL=

POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_PORT=
POSTGRES_HOST=

API42_UID=
API42_PASS=
 ```

### 2. **Start the project**
with the command 'make'
 ```bash
 make
 ```

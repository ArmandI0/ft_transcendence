# ft_transcendence - READ ME


### 1. **Set up the .env file**
prepare and complete the .env file in the folder srcs/secrets :

 ```bash
VAULT_USER_DB=
VAULT_PSWD_DB=

POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=

DJANGO_DB_HOST=
DJANGO_DB_PORT=
DJANGO_DB_NAME=
DJANGO_DB_USER=
DJANGO_DB_PASSWORD=

API42_UID=
API42_PASS=
 ```

### 2. **Start the project**
with the command 'make'
 ```bash
 make
 ```
